# 🦌 Rudolf Schedule (2026 루돌프 스케줄)

**에이전시 프로젝트와 브랜드 리뉴얼 일정을 한눈에 관리하는 프리미엄 스케줄러입니다.** 
상용 수준의 견고함과 미려한 인터페이스를 결합하여 전문적인 프로젝트 관리를 지원합니다.

---

## ✨ 핵심 기능 (Features)
- **Atomic Dashboard**: 중첩된 프로젝트를 격자형 인터페이스로 명확하게 관리.
- **Dynamic Scheduler**: 기간별 드래그 및 레인(Lane) 기반 일정 배정 알고리즘 탑제.
- **Double-Layer Resilience**: `localStorage` 자동 복구 및 에러 바운더리 시스템으로 데이터 안전 보장.
- **Premium UX**: Framer Motion 기반의 매끄러운 전환과 글래스모피즘 디자인.
- **Full Accessibility**: ARIA 표준과 키보드 네비게이션을 완벽하게 지원.

## 🛠️ 기술 스택 (Tech Stack)
- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS 4, Vanilla CSS
- **Icons**: Lucide React
- **Animation**: Framer Motion
- **Date Management**: date-fns
- **Build Tool**: Vite

## 🚀 시작하기 (Getting Started)

### 1. 의존성 설치
```bash
npm install
```

### 2. 개발 서버 실행
```bash
npm run dev
```

### 3. 프로젝트 빌드
```bash
npm run build
```
